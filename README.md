# Deploy - CreSer Marketing App

## Instalar y ejecutar

```bash
cd marketing-app
npm install
npm run dev
```

## Deploy en Vercel

1. Subir a GitHub
2. Ir a https://vercel.com
3. Importar repositorio
4. Deploy automático

## Páginas de la App

| Ruta | Descripción |
|------|-------------|
| `/` | Dashboard con métricas |
| `/generador-contenido` | Generador IA de contenido |
| `/leads` | Gestión de leads |
| `/calendario` | Calendario editorial |
| `/analytics` | Métricas detalladas |
| `/redes-sociales` | Configuración de redes |
| `/configuracion` | Settings generales |

## Características

- ✅ Dashboard con métricas de marketing
- ✅ Generador de contenido con IA
- ✅ Gestión de leads con sentiment analysis
- ✅ Calendario editorial
- ✅ Analytics con gráficos
- ✅ Configuración de redes sociales
- ✅ Configuraciones generales

## Tecnologías

- React 18 + Vite
- Tailwind CSS
- Framer Motion
- Recharts
- Lucide Icons
- Zustand (state)
- date-fns
